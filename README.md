# CollegiateChatter 🎓

An intelligent chatbot system designed for college-related queries, built using Python and Flask.

## 🚀 Project Overview
This project provides a seamless interface for students to get instant answers regarding courses, events, and college information. It utilizes a trained machine learning model to categorize user intents and provide accurate responses.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Machine Learning:** Scikit-learn, NLTK
* **Frontend:** HTML5, CSS3, JavaScript (Bootstrap)
* **Storage:** Git LFS (for large model files)

## 📁 Key Features
* **Intelligent Responses:** Trained on custom college datasets.
* **Interactive UI:** User-friendly chat interface with responsive design.
* **LFS Integration:** Handles large pre-trained model files efficiently.

## 💻 How to Run Locally
1. Clone the repository:
   `git clone https://github.com/Aryakp1/collegiatechatter.git`
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the application:
   `python run.py`
